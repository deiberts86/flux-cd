# FluxCD HomeLab Rancher Cluster
- Home-Lab FluxCD RKE2 Cluster

Core Components:
- Hardened RKE2 nodes on Rocky 9 Linux
- FLUX-CD (bootstrapped)
- RANCHER MCM
- KUBE-VIP
- KUBE-VIP-CLOUD-PROVIDER
- Rancher Local-Path CSI
- RancherBackups (coming soon)
